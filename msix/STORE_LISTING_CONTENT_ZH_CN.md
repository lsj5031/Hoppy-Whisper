## 完整描述(最多 10,000 字符)

```
🎙️ Hoppy Whisper - Windows 闪电级语音转写工具

使用 Hoppy Whisper 即刻将您的语音转换为文字,这是一款注重隐私的 Windows 转写应用,完全离线运行。无需订阅,无需云服务,不收集数据 - 只有快速、准确的语音转文字功能。


✨ 核心功能

🚀 快捷键即时激活
按下 Ctrl+Shift+;(可自定义)开始录音。自然说话,松开即转写,再次点击直接粘贴到任何应用。就是这么简单。

🔒 100% 离线和隐私
所有转写都在您的设备上使用先进的 AI 模型完成。您的语音永远不会离开您的电脑。初始设置后无需互联网连接。

⚡ 极速响应
在现代硬件上不到 1 秒即可完成转写。使用 DirectML 进行 GPU 加速,在 Windows 10/11 上实现最佳性能。

🎯 系统级集成
适用于任何应用 - Word、Excel、Slack、Teams、Outlook、网页浏览器、代码编辑器等。只要能打字的地方,就能使用语音转写。

📜 智能历史记录
自动保存转写历史,默认保留 90 天。使用 Win+Shift+Y 快捷键即时搜索和重用过去的转写。随时导出为 .txt 或 .json。

♿ 无障碍优先
完整的键盘导航、屏幕阅读器支持、高对比度模式和主题感知图标。为每个人设计,包括残障用户。


🎯 适用场景

• 作家和内容创作者 - 免提起草文章、邮件和文档
• 学生 - 记录课堂笔记而不打断专注
• 开发者 - 用语音为代码添加注释
• 商务专业人士 - 快速会议记录和消息起草
• 无障碍用户 - 替代的文字输入方式
• 任何思考速度快于打字速度的人


🔧 工作原理

1. 安装并启动 - 应用静默驻留在系统托盘
2. 首次运行 - 下载 AI 模型(约 500MB,一次性)
3. 按下快捷键 - 默认为 Ctrl+Shift+;(完全可自定义)
4. 自然说话 - 按住快捷键的同时说话
5. 松开 - 不到 1 秒完成转写
6. 粘贴 - 2 秒内再次按下快捷键自动粘贴

就是这样!无需账号,无需注册,没有复杂操作。


🛠️ 技术亮点

• AI 驱动 - 使用优化的 ASR 模型的 ONNX Runtime
• DirectML 加速 - 利用您的 GPU 实现 3-5 倍更快的转写速度
• CPU 回退 - 在任何 x64 Windows PC 上都能良好运行,即使没有独立显卡
• WebRTC VAD - 智能语音活动检测过滤背景噪音
• 低资源占用 - 在托盘中空闲时占用最少的 CPU/内存
• 干净卸载 - 没有注册表污染或残留文件


⚙️ 自定义设置

• 快捷键组合 - 将激活快捷键更改为任何组合
• 粘贴窗口 - 调整 2 秒粘贴延迟(0-5 秒)
• 开机启动 - 登录时自动启动
• 历史记录保留 - 控制转写保存时间(默认 90 天)

所有设置以 JSON 格式本地存储,易于备份和迁移。


🌍 语言支持

当前支持英语转写。多语言模型将在未来更新中推出。


📊 系统要求

最低配置:
• Windows 10(版本 1903 或更高)或 Windows 11
• 64 位处理器(x64)
• 2 GB 内存
• 1 GB 可用磁盘空间(用于模型)
• 麦克风(内置或外置)

推荐配置:
• Windows 11
• 4 GB 或更多内存
• DirectX 12 兼容 GPU(用于硬件加速)
• 优质麦克风以获得最佳准确性

✓ 开源代码可供审计

您的语音属于您。您的文字属于您。您的隐私受到保护。
```
